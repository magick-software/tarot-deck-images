# Tarot Card Decks

This repository contains (high resolution) JPEG images of Tarot card decks.

## Naming Convention

- `m-0` ... `m-21`: major arcana.
- `c-1` .. `c-14`: cups
- `w-1` .. `w-14`: wands
- `p-1` .. `p-14`: pentacles
- `s-1` .. `s-14`: swords
